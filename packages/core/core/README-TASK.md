# Назначение каталога `core`

В каталоге `core` размещаются исходники пакета `camou-core`:

- `__init__.py` — экспорт публичного API (модели и утилиты);
- `models.py` — Pydantic-модели `Runner`, `Session`, `SessionEvent`,
  сопутствующие enum’ы и вспомогательные структуры (proxy/VNC);
- `websocket_bridge.py` — абстракция моста событий и in-memory реализация
  для тестов и локальной разработки.

Модели должны фиксировать статусы сессий и раннеров, типы событий,
инварианты по времени и TTL токенов. Все сущности снабжаются подробными
docstring’ами, описанием полей и примерами.

Если по мере разработки других сервисов потребуется расширить контракт,
вносите изменения аккуратно и с обратной совместимостью — лучше сделать
это один раз, чем часто пересобирать зависимости.
