# Задание: VNC Gateway (FastAPI)

## Цель
Валидировать токены Gateway и проксировать HTTP/WS-соединения к портам VNC Runner’ов.

## Требования
- Эндпоинты: `GET /sessions/{id}` и `/sessions/{id}/ws`.
- Валидация VNC-токена; учёт метрик подключений (можно отложить).

## Качество
- Docstring/inline-комментарии; тесты на валидацию токена (заглушка) и маршрутизацию.
- Обновить `AGENT_NOTES.md`.
