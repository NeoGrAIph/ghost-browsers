# Задание: UI (React + Vite)

## Цель
Собрать операторскую консоль: авторизация Keycloak (PKCE), список сессий (snapshot + SSE), операции (создание/удаление/смена прокси), просмотр VNC.

## Требования
- Состояние: Zustand; данные: React Query. Все сетевые вызовы через `api/client.ts`.
- Auth: keycloak-js; автоматическое обновление токена.
- Надёжный SSE: автопереподключение/backoff.
- Тесты: unit (vitest + RTL) для ключевых сторах/хуков.
